# Mobile FlashCard project for ReactND

## Installation and Launching

- Download source code
  - `git clone git@github.com:doojinkang/reactnd-mobile-flashcards.git`
- Install packages:
  - `cd reactnd-mobile-flashcards; npm install`

- Launch dev server:
  - `npm start`

## Mobile App

Both iOS and Android app runs with Expo.

- iOS
  - Tested under ios simulator
  - Push notification is not tested

- Android
  - Tested under simulator and real device
  - Push notification everyday 8:00PM if user does not study at least one Deck finishes.
